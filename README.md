## Getting Started

### Prerequisites
- Node.js v22.15.0 or later
- Docker
- Modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/GabrielxKuek/dell-innovatedash.git
   cd client
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run with Docker**
   ```bash
   docker-compose up
   ```

4. **Or run locally**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173`
